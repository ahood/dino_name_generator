Contains training code for creating a model to generate dinosaur names
and an interactive notebook, which can be accessed below, to run the
model at the click of a button.

[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/ahood/dino_name_generator/HEAD?urlpath=%2Fvoila%2Frender%2Finteractive_dino_name_generator.ipynb)

Current model is a simple RNN + linear layers and dropout.
